# tiyeni-mobile
tiyeni-mobile

tiyeni-mobile is a React Native + Expo transport app for Malawi, built with Expo Router and Firebase.
It supports driver/passenger workflows, trip creation, location search, and real-time ride management.

🚀 Features

Expo Router navigation (/app folder)
Authentication (login/register)
Driver and Passenger tabs
Trip creation + active trip flow
Location autocomplete + Google-style address handling
Image upload (Cloudinary helper included)
Firebase integration (firebase.ts)
Platform support: Android, iOS, Web
